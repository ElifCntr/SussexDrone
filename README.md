\# SussexDrone



A small-drone detection dataset with bird annotations for the drone-vs-bird discrimination task.



\*\*Status:\*\* Under preparation. Full release planned alongside thesis submission (autumn 2026).



\## Contents



\- 216 video clips across 3 camera types (GoPro, Fujifilm, DJI)

\- Bounding-box annotations for drones and birds

\- Two subsets: detection (for object detection benchmarks) and trajectory (for flight pattern analysis)

\- Train/val/test splits with no session leakage



\## Data access



Dataset will be hosted on Sussex Figshare (DOI to be assigned). This repository contains the conversion scripts and documentation only.
